# 🚀 Generative AI & Data Science Internship – YBI Foundation

This repository contains my learnings, notes, and mini-projects from a 15-day internship at **YBI Foundation** focused on **Generative AI** and **Data Science**. The internship provided hands-on exposure to modern AI techniques and data workflows through curated modules and practical exercises.

---

## 📅 Duration
**15 Days** | June 2025  
**Domain**: Generative AI & Data Science

---

## 🧠 Topics Covered
- Introduction to Artificial Intelligence & Machine Learning
- Python for Data Science
- Pandas & NumPy Essentials
- Exploratory Data Analysis (EDA)
- Data Visualization using Seaborn & Matplotlib
- Introduction to Generative AI
- Prompt Engineering Basics
- Text Generation with Transformers (e.g., GPT-2, GPT-3 Overview)
- Real-world Use Cases of GenAI in DS
